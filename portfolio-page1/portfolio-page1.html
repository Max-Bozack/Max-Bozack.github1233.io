Vacation Website

While making the Vacation Website I had to learn how to use different header sizes. The different size headers you can use are h1,h2,h3,h4,h5, and h6, which progressively get smaller in that order. I learned how to make links between each of my pages, as well as how to insert pictures. Mr. Koch explained my class how to make our websites more vibrant and attractive. Not only did I learn all of this, but I also learned how to make the framework of a website, which was essential to doing other projects in the future.

During this project I was tasked with finding my top five favorite vacation spots, and tell some informaiton about each of them. My top five vacation spots were: Bali, Machu Picchu, Australia, Iceland and Victoria Falls.
