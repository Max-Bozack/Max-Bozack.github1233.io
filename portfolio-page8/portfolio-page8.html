Media Website

After learning what I needed to know to make a basic website, I was now ready to make my own. I was tasked with picking a topic, drawing out the website layout and then making it into a reality. The topic I chose was a popular anime called Jujutsu Kaisen. I had to first, think of what information I wanted to share on the website, before making any physical plans. Once I had some ideas, I drew desgins for three pages, and each one had to be unique. I made a home page, a page about the main heroes of the show, and a page about the villians. Once my plan was approved, I created the website over the next couple weeks.

This was definately my favorite project that I did all year, and I was very pleased with the end result.
